# 📏 Sort Array by String Length

## 📌 Difficulty

**7kyu**

---

## 📝 Problem

Given an array of strings, return the same array sorted from the shortest string to the longest string.

All strings have different lengths, so no tie-breaking is required.

---

## 💡 Approach

- Use JavaScript's built-in `sort()` method.
- Compare the length of each string using a custom comparator.
- Return the sorted array.

---

## ⏱️ Time Complexity

```text
O(n log n)
```

## 💾 Space Complexity

```text
O(1)
```

> JavaScript's `sort()` sorts the array in place.
