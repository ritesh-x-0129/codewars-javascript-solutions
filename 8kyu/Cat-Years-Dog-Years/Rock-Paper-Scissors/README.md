## 📌 Difficulty

**8kyu**

---

## 📝 Problem

Implement the classic **Rock, Paper, Scissors** game.

### Rules

- 🪨 Rock beats Scissors
- ✂️ Scissors beat Paper
- 📄 Paper beats Rock
- Same moves result in a draw

Return:

- `"Player 1 won!"`
- `"Player 2 won!"`
- `"Draw!"`

---

## 💡 Approach

- Check all winning combinations for **Player 1**.
- If both players choose the same move, return **Draw!**
- Otherwise, **Player 2** wins.
