# Function 2 - Squaring an Argument

## 📖 Problem

Write a function that takes a number as an argument and returns its square.

### Example

```javascript
square(7); // 49
```

---

## 💡 Approach

- Multiply the number by itself.
- Return the result.

---

## ⏱️ Complexity

- **Time Complexity:** `O(1)`
- **Space Complexity:** `O(1)`
