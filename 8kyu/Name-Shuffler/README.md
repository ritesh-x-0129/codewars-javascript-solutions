# Name Shuffler

## 📖 Problem

Write a function that swaps the first and last names in a string.

### Example

```javascript
nameShuffler("john McClane"); // "McClane john"
```

---

## 💡 Approach

- Split the string into an array of words.
- Reverse the array.
- Join the words back into a string separated by spaces.

---

## ⏱️ Complexity

- **Time Complexity:** `O(n)`
- **Space Complexity:** `O(n)`
