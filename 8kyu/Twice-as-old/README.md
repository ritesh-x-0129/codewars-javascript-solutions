# 👨👦 Twice as old

## 📌 Difficulty

**8kyu**

---

## 📝 Problem

Given the current age of a father and his son, return the number of years ago (or in the future) when the father was (or will be) exactly twice as old as his son.

The answer is always a non-negative integer.

### Example

```text
dadYearsOld = 36
sonYearsOld = 7

Output:
22
```

---

## 💡 Approach

- Calculate the difference between the father's age and twice the son's age.
- Use `Math.abs()` to ensure the result is always non-negative.
