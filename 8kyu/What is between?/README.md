# What is Between?

## 📖 Problem

Return an array containing all integers between two given numbers (inclusive).

### Example

```javascript
between(1, 4); // [1, 2, 3, 4]
```

---

## 💡 Approach

- Create an empty array.
- Loop from `a` to `b` (inclusive).
- Add each number to the array.
- Return the completed array.

---

## ⏱️ Complexity

- **Time Complexity:** `O(n)`
- **Space Complexity:** `O(n)`
