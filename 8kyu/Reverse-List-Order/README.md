# Question: Reverse List Order

## Difficulty

**8kyu**

---

## Description

Write a function that takes an array and returns a new array with the elements in reverse order.

---

## Example

```javascript
reverseList([1, 2, 3, 4]); // [4, 3, 2, 1]
reverseList([9, 2, 0, 7]); // [7, 0, 2, 9]
```

---

## Approach

- Use JavaScript's built-in `reverse()` method.
- Return the reversed array.

---

## Time Complexity

**O(n)**

## Space Complexity

**O(1)**

> Note: `reverse()` modifies the original array.

---

## Language

JavaScript (ES6+)
